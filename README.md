# PowerBI-US_Walmart
## Este proyecto fue realizado tomando como base un Dataset público en el sitio Kaggle: https://www.kaggle.com/datasets/andrexibiza/grocery-sales-dataset
### Los datos son totalmente ficticios, pero su estructura (previamente transformados) me permitió realizar una serie de análisis y dashboards en PowerBI
### La elección de utilizar Walmart fué para al mismo tiempo poder practicar el darle una correcta identidad de marca al informe.
